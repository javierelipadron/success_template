Contextual classes

Use contextual classes to color table rows or individual cells. You can use this classes putting in the <tr> tag

Type Description
.active Applies the hover color of bootstrap to the rows selected
.success Applies a green color that indicate success
.info Applies a blue color that indicate that the text is informative
.warning Applies a yellow color that indicate to pay atention
.danger Applies a red color that indicate something is not right

Responsive tables

To create responsive tablesby wrapping any .table in .table-responsive. This create a horizontal scroll in the table when the devices are under 768px. When viewing on anything larger than 768px wide, you will not see any difference in these tables.
